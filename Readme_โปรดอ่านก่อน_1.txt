โครงงาน PlasmaX มีเครื่องทั้งหมด 2 ส่วน - เครื่องสายพาน และ เครื่องคัดแยก
โดย Code ต่อไปนี้ในส่วนที่เป็น PlasmaX Machine คือเครื่องคัดแยก

1. Introduction

Source Code ต่อไปนี้สำหรับ โครงการ เครื่องช่วยคัดแยกพลาสม่า PlasmaX 

2. ส่วนประกอบของโค้ด - อันดับแรก โค้ดจะถูกแบ่งออกเป็น 2 folders 

2.1 โค้ดสำหรับ เครื่องคัดแยก - ซึ่งจะใช้ในการวิเคราะห์สีของถุงพลาสม่าและประมวลผลว่าใช้ได้หรือไม่
2.2 โค้ดสำหรับ เว็บไซต์ PlasmaX - คือใช้ Algorithm เดียวกันในการวิเคราะห์สีแบบเครื่องคัดแยก 
แต่ทำการเอาลงเว็บไซต์แทน สามารถใช้ได้จากมือถือและคอมพิวเตอร์
