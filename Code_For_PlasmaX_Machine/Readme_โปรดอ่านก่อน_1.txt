เมื่อเข้ามาใน Folder: Code_For_PlasmaX_Machine นี้จะพบ...

1. Cropped_Photo_Of_Plasma - ภาพของรูปถุงพลาสม่าที่มีการครอปไว้แล้ว
2. หนึ่งในรูปของถุงพลาสม่า ที่ได้ทำการดึงมาจาก Database (จะกล่าวต่อไป)
3. ไฟล์ get_Plama.py
4. ไฟล์ PlamaXCore.py
5. ไฟล์ PlasmaXFinal.py

--------------------------------------------------------------

คำอธิบายแต่ละไฟล์ 

ไฟล์ get_Plasma.py - python file นี้จะทำการดึงรูปของถุงพลาสม่าที่ เครื่องสายพาน ณ โรงบาลศรี ได้ทำการถ่ายภาพไว้
ทำการดึงมาและมา save ไว้ที่ folder เดียวกัน เกิดเป็น หนึ่งในรูปของถุงพลาสม่าในข้อที่ 2 ด้านบน

หนึ่งในรูปของถุงพลาสม่าคือรูปที่ดึงมาจาก database

ไฟล์ PlasmaXCore.py เป็นส่วนของโค้ดที่ใช้ในการประมวลผลของสีพลาสม่า หากใช้ได้จะมี result ออกมาบอกว่า acceptable

ไฟล์ PlamaXFinal.py จะเป็นการรวมเอา get_Plasma.py และ PlasmaXCore.py เข้าด้วยกัน โดยโค้ด 
PlasmaXFinal.py นี้จะทำการดึงรูปมาจาก database ทำการแปลง base64 เป็นรูปภาพ แล้ววิเคราะห์ผล จากนั้นค่อย save ไว้ที่ folder เดิม
โดยสรุป PlasmaXFinal.py จะทำการ ดึง + วิเคราะห์ + save รูปของถุงพลาสม่า
