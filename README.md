# PlasmaX: Automated Blood Plasma Classification System

🏆 **Award: 1st Runner-up** at the 23rd Computer Engineering Project Competition, Khon Kaen University.

## Project Overview
PlasmaX is an intelligent hardware-software integrated system designed for the **Blood Bank of Srinagarind Hospital, Faculty of Medicine, Khon Kaen University**. The system automates the screening process of donated blood plasma, replacing manual visual inspection with high-precision sensors and image processing.

The goal is to identify and categorize plasma into four types:
1. **Normal Plasma**
2. **Lipemic** (High fat content)
3. **Hemolysis** (Red blood cell rupture)
4. **Icteric** (High bilirubin/Jaundice)

## Key Features
- **Real-time Detection:** Utilizes Camera Modules for precise RGB frequency analysis.
- **Image Processing:** Implements OpenCV for plasma area segmentation and noise reduction.
- **Edge Computing:** Optimized for low-power hardware to ensure accessibility for local hospitals.
- **High Accuracy:** Achieved **91.07% accuracy** across 112 clinical test samples.

## System Architecture
The system follows a specific logical workflow:
1. **Input:** Plasma bag placed in a controlled lighting environment.
2. **Sensing:** Sensor captures color data + Camera captures visual features.
3. **Processing:** Thresholding algorithms categorize the plasma based on calibrated RGB values.
4. **Output:** Real-time classification displayed on the interface.
